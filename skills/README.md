# Skills

Vendored agent-authoring skills for building agents on pi-py.

- **[agent-build-pattern](agent-build-pattern/SKILL.md)** — the discipline for
  building a new pi-py agent or subagent: agent-directory layout, `OBJECTIVES.md`,
  a brief `.tau/SYSTEM.md`, extensions with typed Pydantic tool contracts,
  agent-local skills, input/output artifacts, and the prompt-last build sequence
  (compile → unit tests → live evals). See `references/` for the control-plane map
  and the long-form build-sequence rationale.

These are documentation/process guides; they impose no runtime dependency.
